# BulletinBoard-Front-End
React app handle the Bulletin Board Back-End API's

# Steps to run it
- Run the backend first
- Clone from GitHub
- npm install
- Create .env file then fill it with values:
    - REACT_APP_HOST (in development : http://localhost)
    - REACT_APP_PORT (in development : 5000)
- npm run 
- To test it locally with the backend on local install a chrome extension called " Allow-Control-Allow-Origin: * "
